# Ant Design Icons For Vue3

A mobile Ant Design UI Icons library based on Vue 3

## Install

```
npm install ant-mobile-icons-vue3
```

## Usage

```javascript
// ESM 导入
import { CheckCircleFill, SmileOutline } from 'ant-mobile-icons-vue3'

// CommonJS 导入
const { CheckCircleFill } = require('ant-mobile-icons-vue3')

// 单个组件导入
import CheckCircleFill from 'ant-mobile-icons-vue3/es/CheckCircleFill.vue'
```
